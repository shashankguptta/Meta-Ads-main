# Meta Ads Performance Analytics

A data analytics project focused on evaluating digital advertising campaign performance across Facebook and Instagram using SQL, Power BI, and marketing analytics techniques.

## Project Overview

This project analyzes advertising campaigns, user engagement, audience demographics, and targeting effectiveness to identify opportunities for campaign optimization and improved marketing performance.

## Key Objectives

* Analyze campaign performance and engagement metrics
* Compare Facebook and Instagram advertising effectiveness
* Evaluate audience demographics and targeting alignment
* Identify user behavior and conversion patterns
* Generate actionable marketing insights

## Tools & Technologies

* SQL (SQLite)
* Power BI
* Excel Power Query
* Data Visualization
* Marketing Analytics

## Key Insights

* Platform performance varies across engagement and conversion metrics
* Audience demographics significantly influence campaign outcomes
* Time-based user behavior impacts engagement levels
* Targeting mismatches create opportunities for optimization
* Campaign efficiency differs across budget allocations
